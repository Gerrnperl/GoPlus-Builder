# 搜索结果 素材详情页（前端）

## 模块目的

展示选中素材库中素材的详细信息。

## 模块定位

此模块是[素材库页面](./07_frontend_assetLibrary.md)的子模块，在素材库选中公共素材时显示，展示素材详细信息。
